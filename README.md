# 庐江（LuJiang）
---------------
## 项目简述

    此项目成立于2018年1月7日，它是第25条联盟组织下的一个分支项目，主要用来记录庐<br/>
江县教师维权进程中的点点滴滴。
## 背景
**2015、2016和2017年庐江县全体教师没有发放目标考核奖**
